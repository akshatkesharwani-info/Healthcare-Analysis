# Healthcare Data Analysis (Diabetes Dataset)

## Objective
Analyze patient health data to identify key factors associated with diabetes.

## Tools Used
- Python (Pandas, NumPy)
- Seaborn, Matplotlib
- SciPy

## Key Steps
- Data Cleaning and anomaly handling
- Missing value imputation
- Statistical analysis (T-Test)
- Data visualization

## Key Insights
- Glucose is the strongest predictor of diabetes
- BMI and Age also significantly impact diabetes risk
- Correlation analysis confirms strong relationship with glucose

## Conclusion
Glucose, BMI, and Age are critical health indicators.
Early monitoring can help prevent diabetes.
